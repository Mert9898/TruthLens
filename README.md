# TruthLens 🔍

**TruthLens** is an open-source, rules-based misinformation checker for social media content. It helps users analyze posts, tweets, and captions to identify emotional manipulation, suspicious keywords, and questionable sources.

## 🚀 Purpose
In an era of rapid information sharing, TruthLens encourages users to "slow down and think" before hitting the share button. By providing a quick reliability estimate and actionable safety tips, it promotes digital literacy and reduces the spread of misinformation.

## ✨ Core Features
- 📝 **Content Analysis**: Paste any social media text for instant checking.
- ⚖️ **Scoring Engine**: A rule-based system that calculates a reliability score based on linguistics and sources.
- 🚩 **Highlighting**: Identifies manipulative language and suspicious keywords.
- 🔗 **Domain Verification**: Checks links against a curated list of trusted and untrusted sources.
- 💡 **Safety Tips**: Practical advice on how to verify claims manually.
- 📜 **History**: Keep track of your previous analyses.

## 🛠️ Tech Stack
- **Frontend**: React (Modern UI, responsive design)
- **Backend**: FastAPI (Python, high performance)
- **Database**: SQLite (Simple, file-based storage)
- **Styling**: Vanilla CSS (Premium, custom aesthetics)
